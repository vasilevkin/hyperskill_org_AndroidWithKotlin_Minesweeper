Print Username Open on JetBrains Academy

Imagine you are writing a program that greets every new user on some forum. It should output "Hello, NAME!", where NAME is the username. But since it's a forum for hackers, a user can select an option to hide their username during registration. In this situation, the program should output "Hello, secret user!". Try to invoke a greeting function without arguments when the username is hidden.

The input contains a single line which is a username. If the username is hidden, there is "HIDDEN" line.

Output the result of the greeting function.


Sample Input:
Hyperskill
Sample Output:
Hello, Hyperskill!

Sample Input:
HIDDEN
Sample Output:
Hello, secret user!


Memory limit: 256 MB
Time limit: 8 seconds
