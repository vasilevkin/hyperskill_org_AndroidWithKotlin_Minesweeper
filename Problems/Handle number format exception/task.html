Handle number format exception Open on JetBrains Academy

Imagine you implement a web service to pay for video games. There is a text field where a user inputs their card number. A card number is a string like "AAAA AAAA AAAA AAAA", where A is any digit. Please note that a correct card number contains three spaces as in the template. Create a function that returns the card number as a Long or produces any exception when the card number is incorrect.


Sample Input:
1234 5678 9012 3456
Sample Output:
1234567890123456


Memory limit: 256 MB
Time limit: 8 seconds
