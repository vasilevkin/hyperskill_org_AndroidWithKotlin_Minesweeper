Cyclically shifting elements Open on JetBrains Academy

Write a program that reads an array of ints named A and cyclically shifts the elements of the array to the right: A[0] goes to the place of A[1], A[1] takes the place of A[2], ..., and the last element goes to the place of A[0]).

The first line of the input contains the number of elements in the array. The second line is the elements of the array.

The output contains all the shifted elements of the array. Separate the elements by the space character.


Sample Input:
5
1 2 3 4 5
Sample Output:
5 1 2 3 4


Memory limit: 256 MB
Time limit: 1 second
