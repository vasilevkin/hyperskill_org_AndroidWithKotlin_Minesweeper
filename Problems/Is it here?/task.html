Write a program that checks if a set of N numbers contains some number M.

 

The first line contains the number N.

The second line contains N numbers separated by space.

The third line contains one integer number M.

 

Output "YES" or "NO".


Sample Input:
3
1 3 2
3

Sample Output:
YES

Sample Input:
5
1 3 2 5 4
7

Sample Output:
NO


Memory limit: 256 MB
Time limit: 8 seconds
